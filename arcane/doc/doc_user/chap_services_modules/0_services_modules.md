# Available Services and Modules {#arcanedoc_services_modules}

This chapter presents the services and modules available in %Arcane.  
These modules and services can be used in %Arcane codes.  
It is recommended that you have read the chapter \ref arcanedoc_core_types and
more specifically the section
\ref arcanedoc_core_types_axl_caseoptions_options_service to understand how to
include a service or a module in your code.

<br>
