# Manuel développeur de Sample {#sampledoc_devmanual}

\mainpage

<!-- L'ordre des subpages est important ici. -->

Cette page contient la documentation pour les développeurs de %Sample.

La page \subpage sampledoc_news présente la liste des nouveautés de chaque version.

La documentation accessible depuis cette page s'adresse aux développeurs de %Sample.
Elle contient plus de documentations de classes et est plus
exhaustive que la documentation utilisateur. Elle est aussi plus longue à générer
et plus volumineuse.

La documentation utilisateur se compose de chapitres, sous-chapitres et de pages.
La documentation développeur est plus libre et peut être composé uniquement de pages
(même si la disposition présentée dans la page \subpage sampledoc_doxygen est souhaitable
dans les deux documentations).

<br>

Les pages/chapitres disponibles sont:

- \ref sampledoc_doxygen <br>
  Présente comment contribuer à la documentation %Sample.

- \subpage sampledoc_services_modules <br>
  Présente les services et les modules mis à disposition des développeurs
