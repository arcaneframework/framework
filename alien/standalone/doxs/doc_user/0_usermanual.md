# %Alien User Manual {#aliendoc_usermanual}

\mainpage %Alien

This page contains the documentation for %Alien users.

The documentation accessible from this page is intended for individuals
who wish to learn about %Alien’s features.

The documentation is organized into chapters (listed below), which are further divided
into subchapters and/or pages.

Each chapter/subchapter includes a summary briefly describing its content
and a list of its pages.

<br>

The available chapters are:

- \subpage aliendoc_build_install <br>
  Building and Using %Alien.

- \subpage aliendoc_concepts <br>
  %Alien’s High-Level Design.

- \subpage aliendoc_tutorials <br>
  Tutorials for using %Alien with three APIs.

- \subpage aliendoc_sycl <br>
  Covers how to use %Alien with SYCL.
